# Architecture decision records

This directory is the canonical home for finalized system-wide architecture decisions.

Context-specific decisions live beside their context under `apps/<context>/docs/adr/` or `packages/<context>/docs/adr/`.

Exploration and evolving notes remain in the Niama Obsidian vault. Link the relevant Obsidian note and Linear issue from each ADR instead of duplicating their contents.
